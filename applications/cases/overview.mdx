## 常見實用案例

這些是最常被使用和詢問的案例：

<AccordionGroup>
  <Accordion title="如何生成去背圖" icon="scissors">
    **需求**：生成已經去背的素材圖片
    
    **工具**：ChatGPT
    
    **核心技巧**：在提示詞中明確說明「白色背景」或「透明背景」
    
    ```
    請生成一個咖啡杯的產品圖，
    純白色背景，物品完全置中，
    俯視視角，專業產品攝影風格
    ```
    
    [查看詳細教學 →](/ai-tools/chatgpt/practical-cases#去背圖)
  </Accordion>
  
  <Accordion title="生成漫畫/插圖" icon="book-open">
    **需求**：創作插畫風格或漫畫風格的圖片
    
    **工具**：ChatGPT, 即夢
    
    **核心技巧**：明確指定風格類型和視覺元素
    
    ```
    請生成一個日本漫畫風格的角色插圖：
    可愛的貓咪擬人化，穿著咖啡師圍裙，
    在咖啡廳工作，大眼睛，Q 版比例，
    簡潔的線條，淡彩上色
    ```
    
    [查看詳細教學 →](/ai-tools/chatgpt/practical-cases#漫畫)
  </Accordion>
  
  <Accordion title="如何生成一組 icon" icon="icons">
    **需求**：生成風格一致的系列 icon
    
    **工具**：ChatGPT, Recraft
    
    **核心技巧**：一次描述整組的統一風格，列出所有需要的 icon
    
    ```
    請生成一組 6 個天氣 icon：
    
    風格：3D 立體、圓潤可愛、漸層配色（藍色系）
    背景：統一白色背景
    
    包含：晴天、多雲、雨天、雪天、雷雨、大風
    ```
    
    [查看詳細教學 →](/ai-tools/chatgpt/practical-cases#icon組)
  </Accordion>
  
  <Accordion title="產品照片生成不同場景" icon="camera">
    **需求**：同一個產品放在不同場景中
    
    **工具**：FLORA, Flair AI, ChatGPT
    
    **核心技巧**：保持產品一致性，改變場景描述
    
    ```
    場景 1：木質桌面，自然光，咖啡廳氛圍
    場景 2：大理石檯面，明亮光線，現代廚房
    場景 3：戶外草地，陽光灑落，野餐情境
    ```
    
    [查看詳細教學 →](/ai-tools/flora/product-scenes)
  </Accordion>
  
  <Accordion title="貼圖生成" icon="sticky-note">
    **需求**：製作社群貼圖或表情貼
    
    **工具**：ChatGPT
    
    **核心技巧**：統一風格，明確表情和動作
    
    ```
    請生成一組 8 個可愛熊貓表情貼圖：
    
    風格：Q 版卡通、簡單線條、白色背景
    表情：開心、難過、生氣、驚訝、
          睡覺、思考、大笑、害羞
    ```
    
    [查看詳細教學 →](/ai-tools/chatgpt/practical-cases#貼圖)
  </Accordion>
</AccordionGroup>

## 學習建議

<Steps>
  <Step title="從簡單開始">
    選擇一個基礎案例，完整跟著做一遍
  </Step>
  
  <Step title="理解原理">
    不要只是複製提示詞，理解為什麼這樣寫
  </Step>
  
  <Step title="修改調整">
    根據自己的需求修改提示詞和參數
  </Step>
  
  <Step title="建立資料庫">
    保存好用的提示詞和案例供日後使用
  </Step>
  
  <Step title="持續實踐">
    在實際專案中應用，累積經驗
  </Step>
</Steps>

<Tip>
**專業建議**

建立自己的案例筆記，記錄：
- 使用的提示詞
- 生成結果的優缺點
- 後續如何調整
- 適合的應用場景
</Tip>
