## 快速翻譯技巧

不知道大家平時如何使用翻譯的，是每次要翻譯都打開 Google 翻譯之類的工具或是現在使用 ChatGPT 之類的 AI 來翻譯呢？

這次想分享的是比上面這二種方法都還快速，且品質很好的翻譯方法。

<Note>
**使用工具**

Raycast 的 Deepcast（Mac 用戶專用）
</Note>

## 安裝與設定步驟

<Steps>
  <Step title="安裝 Raycast">
    前往 [Raycast 官網](https://www.raycast.com/)下載並安裝
    
    Raycast 是 Mac 上強大的生產力工具，可以替代 Spotlight，提供更多實用功能。
  </Step>

  <Step title="安裝 Deepcast 擴充套件">
    在 Raycast Store 搜尋並安裝 [Deepcast](https://www.raycast.com/mooxl/deepcast)
    
    這是 Raycast 的 DeepL 翻譯擴充套件。
  </Step>

  <Step title="註冊 DeepL 帳號">
    前往 [DeepL Pro](https://www.deepl.com/pro) 註冊免費帳號
    
    <Tip>
    DeepL 是網路評價非常好的翻譯工具，免費版每月提供 **50 萬字元**的翻譯額度，一般個人使用非常足夠。
    </Tip>
  </Step>

  <Step title="登入並創建 API 密鑰">
    登入後前往 [DeepL 帳號頁面](https://www.deepl.com/account/summary)
    
    在「API 密鑰」區域點選「創建密鑰」
    
    <Warning>
    **重要提醒**
    
    - 記得妥善保管 API 密鑰，不要分享給他人
    - 所有翻譯用量都會從你的帳號扣除
    - 建議將密鑰儲存在安全的地方
    </Warning>
  </Step>

  <Step title="設定 Raycast 擴充套件">
    **設定 API 金鑰：**
    
    1. 打開 Raycast 的 Settings
    2. 選擇「Extensions」
    3. 找到「Deepcast」
    4. 在右側的「API Key」欄位貼上你的 API 金鑰
    
    **設定翻譯語言與快捷鍵：**
    
    1. 選擇目標語言（例如：English (US)）
    2. 設定快捷鍵（建議：`Cmd + E`）
    
    <Note>
    快捷鍵可以自訂，建議選擇好記且不會與其他工具衝突的組合。
    </Note>
  </Step>

  <Step title="開始使用">
    設定完成後，在任何應用程式中：
    
    1. 選取要翻譯的文字
    2. 按下設定的快捷鍵（例如：`Cmd + E`）
    3. 翻譯結果會自動複製到剪貼簿
    4. 直接貼上使用
    
    成功翻譯後會在畫面頂部顯示綠色通知訊息。
  </Step>
</Steps>

### 1. 圖庫搜尋應用

當你在 Unsplash、Pexels 等圖庫網站搜尋素材時：

```
原本流程：
1. 想搜尋「夕陽海邊」
2. 打開翻譯工具
3. 輸入「夕陽海邊」
4. 複製翻譯結果 "sunset beach"
5. 貼到圖庫搜尋框

使用 Raycast + DeepL：
1. 在圖庫搜尋框直接輸入「夕陽海邊」
2. Cmd + A + Cmd + E
3. 完成！立即搜尋
```

<Tip>
**時間節省：從 30 秒縮短到 3 秒**
</Tip>

### 2. AI 圖片生成應用

在使用 ChatGPT、Midjourney、Stable Diffusion 等工具時：

```
傳統方式：
1. 想到創意「一隻戴墨鏡的貓在海邊度假」
2. 手動翻譯成英文提示詞
3. 複製貼上到 AI 工具

使用 Raycast + DeepL：
1. 直接在提示詞輸入框打中文
2. Cmd + A + Cmd + E
3. 直接生成！
```

<Note>
**為什麼 AI 生成要用英文？**

雖然現在很多 AI 工具支援中文，但英文提示詞通常能得到：
- 更精確的結果
- 更豐富的細節
- 更好的風格控制
</Note>

### 3. 設計軟體應用

在 Figma、Sketch、Adobe XD 等設計軟體中：

```
情境：需要在設計稿中加入英文文案

傳統方式：
1. 在文字工具中輸入中文
2. 打開翻譯工具
3. 翻譯後複製
4. 回到設計軟體貼上

使用 Raycast + DeepL：
1. 在文字圖層直接打中文
2. 選取後 Cmd + E
3. 直接貼上替換
```

## 進階技巧

<AccordionGroup>
  <Accordion title="設定多種語言快捷鍵" icon="language">
    如果你經常需要翻譯成不同語言，可以設定多組快捷鍵：
    
    - `Cmd + E`：翻譯成英文
    - `Cmd + J`：翻譯成日文
    - `Cmd + K`：翻譯成韓文
    
    在 Raycast Extensions 設定中，可以為同一個擴充套件設定多個指令。
  </Accordion>

  <Accordion title="與 Clipboard Manager 搭配" icon="clipboard">
    安裝 Raycast 的剪貼簿管理器：
    
    1. 所有翻譯結果都會自動儲存在剪貼簿歷史
    2. 可以快速回顧之前的翻譯
    3. 需要時隨時取用
  </Accordion>

  <Accordion title="翻譯長文本" icon="file-lines">
    DeepL 支援翻譯長文本，但要注意：
    
    - 免費版每月 50 萬字元額度
    - 超過後需要升級付費方案
    - 可以在 DeepL 帳號查看使用量
  </Accordion>

  <Accordion title="自訂翻譯方向" icon="arrows-left-right">
    在 Deepcast 設定中可以：
    
    - 設定預設翻譯方向（中→英、英→中）
    - 開啟自動偵測語言
    - 保留原文格式
  </Accordion>
</AccordionGroup>

## 常見問題

<AccordionGroup>
  <Accordion title="快捷鍵沒有反應？" icon="keyboard">
    **可能原因：**
    
    1. 快捷鍵與其他應用衝突
       - 檢查系統偏好設定中的快捷鍵
       - 嘗試更換其他組合
    
    2. Raycast 沒有取得輔助功能權限
       - 前往「系統偏好設定」→「隱私權與安全性」→「輔助功能」
       - 確認 Raycast 已勾選
    
    3. 擴充套件未正確安裝
       - 重新安裝 Deepcast 擴充套件
  </Accordion>

  <Accordion title="翻譯結果不準確？" icon="circle-question">
    **改善方法：**
    
    1. 確保選取完整的句子或段落
    2. 避免選取包含特殊符號或格式的文字
    3. 如果是專業術語，可能需要手動調整
    4. 考慮使用 ChatGPT 進行更專業的翻譯
  </Accordion>

  <Accordion title="API 額度用完了怎麼辦？" icon="battery-empty">
    **解決方案：**
    
    1. 升級到 DeepL Pro 付費方案
    2. 等待下個月額度重置
    3. 短期內可使用其他翻譯工具
    4. 在 DeepL 帳號中可查看使用量
  </Accordion>

  <Accordion title="Windows 用戶可以使用嗎？" icon="windows">
    Raycast 目前僅支援 macOS。Windows 用戶可以考慮：
    
    - PowerToys（微軟官方工具）
    - AutoHotkey（自動化腳本）
    - DeepL 桌面應用程式
    - 瀏覽器擴充套件
  </Accordion>
</AccordionGroup>
