## Prompt 資源導覽

好的 Prompt（提示詞）是生成優質 AI 圖片的關鍵。本頁精選了多個優質的 Prompt 資源網站，幫助你學習如何撰寫更有效的提示詞。

<Note>
**什麼是 Prompt？**

Prompt 是你給 AI 的指令或描述，告訴 AI 你想要生成什麼樣的圖片。好的 Prompt 包含：
- 主題描述
- 風格指定
- 構圖細節
- 色調氛圍
- 技術參數
</Note>

## 如何有效使用 Prompt 資源

<AccordionGroup>
  <Accordion title="1. 建立 Prompt 資料庫" icon="database">
    **整理方式：**
    
    - 使用 Notion 或 Evernote
    - 按主題分類（人物、風景、產品等）
    - 按風格分類（寫實、插畫、抽象等）
    - 標註來源和效果
    
    **建議結構：**
    ```
    📁 我的 Prompt 資料庫
    ├── 📂 人物攝影
    │   ├── 人像特寫
    │   ├── 全身照
    │   └── 群體照
    ├── 📂 產品設計
    │   ├── 電子產品
    │   ├── 食品飲料
    │   └── 服飾配件
    └── 📂 風格特效
        ├── 賽博龐克
        ├── 復古懷舊
        └── 極簡現代
    ```
  </Accordion>
  
  <Accordion title="2. 學習拆解 Prompt" icon="magnifying-glass">
    **分析結構：**
    
    優秀的 Prompt 通常包含：
    
    1. **主體描述** (30%)
       - 是什麼物件/人物/場景
       - 主體的狀態和動作
    
    2. **風格指定** (25%)
       - 藝術風格
       - 攝影風格
       - 視覺風格
    
    3. **環境設定** (20%)
       - 背景描述
       - 光線條件
       - 氛圍營造
    
    4. **技術細節** (15%)
       - 視角和構圖
       - 景深和焦距
       - 色彩處理
    
    5. **品質控制** (10%)
       - 解析度要求
       - 渲染品質
       - 細節程度
    
    **練習：**
    拿一個好的 Prompt，嘗試標註每個部分的作用。
  </Accordion>
  
  <Accordion title="3. 漸進式學習" icon="chart-line">
    **學習路徑：**
    
    **階段一：模仿（1-2 週）**
    - 直接使用現有 Prompt
    - 觀察生成結果
    - 理解各部分功能
    
    **階段二：調整（2-4 週）**
    - 修改部分內容
    - 替換主題元素
    - 調整風格描述
    - 比較結果差異
    
    **階段三：創造（持續）**
    - 自己撰寫 Prompt
    - 結合多個技巧
    - 發展個人風格
    - 建立模板庫
  </Accordion>
  
  <Accordion title="4. A/B 測試" icon="flask">
    **測試方法：**
    
    1. **單變數測試**
       - 只改變一個元素
       - 觀察結果變化
       - 記錄有效改變
    
    2. **風格測試**
       ```
       原始：a modern office interior
       測試A：a minimalist modern office interior
       測試B：a cozy modern office interior
       測試C：a futuristic modern office interior
       ```
    
    3. **參數測試**
       - 調整權重
       - 改變比例
       - 測試參數組合
    
    4. **記錄結果**
       - 截圖保存
       - 標註差異
       - 總結規律
  </Accordion>
</AccordionGroup>

## 不同 AI 工具的 Prompt 差異

<Tabs>
  <Tab title="Midjourney">
    **特色：**
    - 重視藝術風格描述
    - 參數系統豐富
    - 擅長創意和藝術性圖片
    
    **Prompt 技巧：**
    ```
    /imagine prompt: [主題], [風格], [構圖], 
    [光線], [色調] --ar 16:9 --v 6
    ```
    
    **常用參數：**
    - `--ar` 比例
    - `--v` 版本
    - `--s` 風格化程度
    - `--c` 混亂度
  </Tab>
  
  <Tab title="DALL-E / ChatGPT">
    **特色：**
    - 理解自然語言
    - 適合描述性指令
    - 擅長特定主題和產品
    
    **Prompt 技巧：**
    ```
    A detailed description of [主題] 
    in [風格] style, with [特徵], 
    [環境], [光線], high quality
    ```
    
    **建議：**
    - 使用完整句子
    - 描述越詳細越好
    - 可以對話式調整
  </Tab>
  
  <Tab title="Stable Diffusion">
    **特色：**
    - 高度可控
    - 支援負面提示
    - 適合精確控制
    
    **Prompt 技巧：**
    ```
    正面提示：
    [主題], [風格], [品質詞], 
    [技術細節]
    
    負面提示：
    [不要的元素], low quality, 
    blurry, distorted
    ```
    
    **進階技巧：**
    - 使用權重 (word:1.5)
    - 混合概念 [A:B:0.5]
    - LoRA 模型
  </Tab>
</Tabs>

## 常見錯誤與解決

<Warning>
**避免這些常見錯誤**

1. **過度簡短**
   - ❌ "a cat"
   - ✅ "a fluffy orange tabby cat sitting on a windowsill, golden hour lighting, cozy atmosphere"

2. **過度複雜**
   - ❌ 一次描述太多元素導致混亂
   - ✅ 聚焦主題，適度添加細節

3. **矛盾描述**
   - ❌ "dark bright scene"（矛盾）
   - ✅ "dimly lit scene with bright highlights"（清楚）

4. **忽略品質詞**
   - ❌ 只描述內容，不指定品質
   - ✅ 加上 "high quality, detailed, professional"
</Warning>
