## 情境一：讓素材更符合需求

### 使用場景

當你找到的素材不完全符合專案需求時，可以用 Nano Banana 來調整和優化。

### 實際案例

**需求**：需要一張符合品牌調性的產品情境圖

```
原始素材：一張普通的咖啡杯照片
需求：要符合品牌「溫暖、自然、手工感」的調性

請幫我調整這張咖啡杯照片：
- 背景改為木質桌面
- 增加溫暖的自然光線
- 旁邊添加手工製作的元素
- 整體色調偏向暖色系
- 營造手工咖啡店的氛圍
```

<Tip>
**專業秘訣**

先描述你想要的氛圍和感覺，再具體說明需要調整的元素，這樣能得到更符合需求的結果。
</Tip>

## 情境二：生成裝飾用素材

### 使用場景

需要各種裝飾性元素來豐富設計，如背景紋理、裝飾圖案、邊框等。

### 實際案例

**需求**：為簡報製作裝飾性背景元素

```
請生成一組裝飾性素材：

1. 幾何圖形背景
- 簡潔的線條圖案
- 藍色和白色為主
- 適合簡報使用

2. 自然紋理背景
- 紙張或布料質感
- 溫暖的米色調
- 微妙的紋理細節

3. 科技感裝飾
- 發光線條和點
- 深色背景
- 未來感設計
```

<Note>
裝飾素材通常需要保持簡潔，避免搶奪主要內容的注意力。
</Note>

## 情境三：漫畫與插圖

### 使用場景

需要各種風格的插圖和漫畫，用於網站、App、簡報或行銷素材。

### 實際案例

**需求**：為健康 App 製作引導插圖

```
請生成一組健康主題的插圖：

風格：簡潔的線條插畫，現代扁平化設計
色調：清新的綠色和藍色系

內容：
1. 運動的人 - 跑步或瑜伽姿勢
2. 健康飲食 - 蔬果和餐盤
3. 充足睡眠 - 月亮和床鋪
4. 心情放鬆 - 冥想或深呼吸

要求：每個插圖都要傳達正面積極的感覺
```

<Tip>
**風格一致性**

在生成系列插圖時，要明確指定統一的風格、色調和構圖方式，確保視覺一致性。
</Tip>

## 情境四：延伸圖庫素材

### 使用場景

當你找到一張不錯的圖庫素材，但需要更多變化或不同角度的版本時。

### 實際案例

**需求**：基於現有素材生成系列變化**

```
我有一張咖啡廳的室內照片，請幫我生成幾個不同的版本：

原始素材：現代簡約風格的咖啡廳
需求變化：
1. 同樣空間，但改為夜晚氛圍
2. 同樣空間，但改為復古風格
3. 同樣空間，但改為工業風
4. 同樣空間，但改為自然植物風

要求：保持空間結構相似，但氛圍和風格完全不同
```

<Note>
延伸圖庫素材時，要明確說明要保持哪些元素不變，哪些元素要調整，這樣能得到更一致的系列作品。
</Note>

## 情境五：生成 YouTube 封面

### 使用場景

需要快速製作吸引人的 YouTube 影片封面，提升點擊率。

### 實際案例

**需求**：為「AI 設計教學」系列影片製作封面**

```
請生成一張 YouTube 封面圖：

主題：「用 ChatGPT 設計 Logo」
尺寸：1280x720 橫向

視覺元素：
- 中央放置一個簡潔的 Logo 設計
- 背景使用漸層色彩（藍色到紫色）
- 添加一些設計工具圖示（筆、尺、電腦）
- 標題區域預留空間

風格：現代簡約，專業感
色調：科技感配色，適合設計主題
```

<Tip>
**YouTube 封面設計要點**

- 確保在小尺寸下也能清楚看到主要元素
- 使用對比強烈的色彩
- 標題文字要簡潔有力
- 預留標題和頻道名稱的空間
</Tip>

## 情境六：IP 衍生應用

### 使用場景

為品牌 IP 或角色設計延伸應用，如系列海報、周邊商品、行銷素材等。

### 實際案例

**需求**：為品牌吉祥物設計系列海報**

```
我有一個可愛的熊貓吉祥物，請幫我設計一組系列海報：

吉祥物：圓潤可愛的熊貓，穿著圍裙
品牌調性：溫暖、親切、專業

系列主題：
1. 「歡迎光臨」- 熊貓在門口迎接
2. 「品質保證」- 熊貓在檢查產品
3. 「貼心服務」- 熊貓在協助客戶
4. 「感謝支持」- 熊貓鞠躬致謝

風格：溫馨插畫風格，暖色調
要求：保持角色一致性，但動作和場景不同
```

<Note>
**IP 應用實驗**

參考：[用 AI 設計了系列海報(實驗)](https://creatorhome.gitbook.io/ai-design/chang-yong-ai-she-ji-gong-ju-zhi-nan/chatgpt-tu-pian-sheng-cheng-ying-yong/she-ji-shi-de-7-zhong-ying-yong-an-li)

IP 衍生應用可以快速測試不同的視覺方向，找到最適合品牌調性的設計風格。
</Note>

## 情境七：搭配 AI 工具做動畫

### 使用場景

結合 Nano Banana 生成的靜態圖片和其他 AI 工具，製作簡單的動畫效果。

### 實際案例

**需求**：為社群媒體製作動態貼文**

```
步驟 1：用 Nano Banana 生成基礎圖片
請生成一張適合動畫的插圖：
- 一個可愛的咖啡杯
- 簡潔的線條風格
- 白色背景
- 預留動畫空間

步驟 2：描述動畫需求
請幫我設計這個咖啡杯的動畫效果：
- 咖啡杯輕微搖擺
- 咖啡表面有漣漪效果
- 背景有溫暖的光暈變化
- 整體氛圍溫馨舒適
```

<Tip>
**AI 工具組合使用**

- Nano Banana：生成基礎圖片和動畫概念
- Runway ML：製作動畫效果
- CapCut：後製和剪輯
- Canva：添加文字和特效

多個 AI 工具結合使用，可以創造更豐富的視覺效果。
</Tip>
