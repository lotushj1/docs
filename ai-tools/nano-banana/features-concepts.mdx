## Nano Banana 的基本功能

### 1. 文字生成圖片

最基本的功能就是用文字描述生成圖片。

<Steps>
  <Step title="描述你的需求">
    用自然語言告訴 Nano Banana 你想要什麼圖片
  </Step>
  
  <Step title="AI 理解並優化">
    Nano Banana 會理解你的需求，並優化提示詞
  </Step>
  
  <Step title="生成圖片">
    後台的 Gemini 2.5 Flash Image Preview 引擎會生成圖片
  </Step>
  
  <Step title="查看結果">
    圖片會直接顯示，可以繼續對話調整
  </Step>
</Steps>

**範例提示詞：**

```
請幫我生成一張產品攝影風格的圖片：
一罐咖啡豆放在木質桌面上，旁邊有散落的咖啡豆，
背景有柔和的自然光線，整體氛圍溫暖舒適
```

### 2. 圖片編輯功能

Nano Banana 還可以編輯現有圖片（包括上傳的圖片或生成的圖片）。

<Tabs>
  <Tab title="局部修改">
    選擇圖片的特定區域進行修改
    
    ```
    請把這張圖片中的咖啡杯改成透明玻璃杯
    ```
  </Tab>
  
  <Tab title="添加元素">
    在圖片中增加新的物件
    
    ```
    在這張圖的右上角添加一些綠色植物
    ```
  </Tab>
  
  <Tab title="移除元素">
    去除圖片中不需要的部分
    
    ```
    請移除背景中的那個人
    ```
  </Tab>
  
  <Tab title="風格調整">
    改變整體視覺風格
    
    ```
    把這張照片轉換成水彩畫風格
    ```
  </Tab>
</Tabs>

### 3. 擴圖功能

可以擴展圖片的邊界，生成原本沒有的區域。

<Note>
**使用場景**

- 橫向圖片需要改成直向
- 需要更多背景空間
- 調整構圖比例
- 製作不同尺寸版本
</Note>

## 核心概念理解

### 提示詞的重要性

<AccordionGroup>
  <Accordion title="具體 vs 模糊" icon="magnifying-glass">
    **❌ 模糊**
    ```
    一個美麗的風景
    ```
    
    **✅ 具體**
    ```
    日落時分的海邊，柔和的橘色天空，
    平靜的海面反射著夕陽，
    前景有幾顆棕櫚樹的剪影
    ```
  </Accordion>
  
  <Accordion title="包含關鍵元素" icon="list">
    好的提示詞應該包含：
    - **主體**：圖片的主要內容
    - **環境**：場景和背景
    - **光線**：光線方向和氛圍
    - **風格**：視覺風格和調性
    - **視角**：拍攝角度
    - **色彩**：主要色調
  </Accordion>
  
  <Accordion title="使用參考詞彙" icon="book">
    使用專業術語能得到更好的結果：
    - **攝影術語**：淺景深、逆光、黃金時刻
    - **設計風格**：極簡、復古、賽博龐克
    - **藝術流派**：印象派、波普藝術
    - **材質描述**：磨砂、光澤、透明
  </Accordion>
</AccordionGroup>

### 尺寸與比例

Nano Banana 支援生成不同比例的圖片：

<CardGroup cols={3}>
  <Card title="方形" icon="square">
    **1:1 比例**
    
    適合：Instagram 貼文、頭像、圖示
  </Card>
  
  <Card title="橫向" icon="rectangle-wide">
    **16:9 或 4:3**
    
    適合：網站 Banner、YouTube 封面、簡報
  </Card>
  
  <Card title="直向" icon="rectangle-vertical">
    **9:16 或 3:4**
    
    適合：限時動態、Pinterest、手機螢幕
  </Card>
</CardGroup>

**指定尺寸範例：**

```
請生成一張 1792x1024 的橫向圖片
或
請生成一張適合 Instagram 限時動態的直向圖片
```

## 使用限制與注意事項

<Warning>
**重要限制**

1. **版權問題**
   - 不能要求生成特定藝術家或攝影師的風格
   - 不能生成知名品牌 Logo
   - 避免使用受版權保護的角色

2. **內容限制**
   - 不能生成暴力、色情、政治敏感內容
   - 不能生成真實人物的肖像
   - 某些敏感詞彙會被過濾

3. **技術限制**
   - 文字生成容易出錯
   - 複雜的手部細節可能不準確
   - 極度詳細的要求可能無法完全實現
</Warning>

<Tip>
**專業建議**

- 先用 Nano Banana 生成基礎圖片
- 再用專業軟體（Photoshop）進行精修
- 保存好用的提示詞模板
- 利用 Nano Banana 的對話特性持續優化
</Tip>

## 實用技巧總結

### 提示詞撰寫公式

一個好的提示詞通常遵循這個結構：

```
[主體] + [動作/狀態] + [環境/背景] + [光線] + [風格] + [視角] + [細節]
```

**範例：**

```
一位年輕女性，微笑著看向鏡頭，站在現代咖啡廳內，
溫暖的自然光從窗戶灑進，極簡現代風格，
正面拍攝，景深淺，背景柔焦
```

### 迭代優化流程

<Steps>
  <Step title="第一次生成">
    使用基本描述生成初版圖片
  </Step>
  
  <Step title="評估結果">
    找出需要改進的地方
  </Step>
  
  <Step title="具體調整">
    針對具體問題進行修改
    - 「讓光線更柔和」
    - 「把背景改成純色」
    - 「增加一些細節」
  </Step>
  
  <Step title="持續優化">
    重複評估和調整，直到滿意
  </Step>
</Steps>

## 常見問題解答

<AccordionGroup>
  <Accordion title="為什麼生成的圖片不符合預期？" icon="circle-question">
    **可能原因：**
    1. 描述不夠具體
    2. 要求太複雜
    3. 使用了模糊的詞彙
    
    **解決方法：**
    - 增加具體的描述細節
    - 分步驟進行修改
    - 參考成功的提示詞範例
  </Accordion>
  
  <Accordion title="如何讓圖片更真實？" icon="image">
    **技巧：**
    - 加入「攝影風格」、「真實照片」等關鍵字
    - 描述光線細節（自然光、柔光、逆光等）
    - 說明材質和紋理（磨砂、光澤、粗糙等）
    - 指定相機參數（淺景深、bokeh 效果等）
  </Accordion>
  
  <Accordion title="生成的文字為什麼會錯誤？" icon="font">
    **原因：**
    AI 圖片生成對文字的處理仍然不夠完美
    
    **解決方案：**
    1. 避免在圖片中生成複雜文字
    2. 簡短的文字（1-2 個字）成功率較高
    3. 重要文字建議後期用 Photoshop 添加
    4. 多次生成選擇最好的結果
  </Accordion>
  
  <Accordion title="如何控制圖片的色調？" icon="palette">
    **方法：**
    
    在提示詞中明確指定色調：
    - 「溫暖色調」、「冷色調」
    - 「柔和的粉色系」
    - 「高對比度黑白」
    - 「復古褪色效果」
    
    **範例：**
    ```
    請生成一張溫暖色調的咖啡廳照片，
    以橙色和褐色為主，營造舒適放鬆的氛圍
    ```
  </Accordion>
</AccordionGroup>
