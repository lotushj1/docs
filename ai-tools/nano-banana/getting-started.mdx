## Nano Banana 是什麼？

Nano Banana（奈米香蕉）是 Google 最新推出的 **Gemini 2.5 Flash Image Preview** 模型的代號。這個名字的由來相當有趣，也展現了 Google 創新的行銷策略。

<Note>
**有趣的命名故事**

Google 將這個模型以「Nano Banana」的匿名身份投入公開的 AI 圖像生成平台（LMArena）進行盲測競賽。憑藉超強的圖像品質，它迅速在社群中引發熱烈討論，並榮登排行榜首位。

這套「先用實力征服社群，再揭曉身份」的策略，成功為新模型創造了巨大的口碑與話題性，是一次非常成功的行銷案例。
</Note>

### 核心定位

- **整合型工具**：整合在 Gemini 中的專業級圖像生成與編輯模型
- **API 支援**：提供 API 串接，可導入各種 AI 工具
- **應用導向**：從「單張驚艷」走向「連續應用」的實用工具

## 新手第一步：從這裡開始！

<Steps>
  <Step title="上傳自己的照片">
    選擇一張清晰的人物照片作為起點
  </Step>
  
  <Step title="嘗試簡單指令">
    **推薦指令：**
    - 「讓他笑」
    - 「換個髮型」
    - 「改成卡通風格」
    - 「換個背景」
    
    用最簡單的指令親身感受 Nano Banana 的威力！
  </Step>
  
  <Step title="觀察結果">
    注意觀察：
    - 人物特徵是否保持一致
    - 光影處理是否自然
    - 細節是否真實
  </Step>
  
  <Step title="進階嘗試">
    熟悉基本操作後，可以嘗試：
    - 更複雜的修改
    - 多次連續編輯
    - 風格轉換
    - 場景變化
  </Step>
</Steps>

<Tip>
**重要提示**

Nano Banana 的理解能力超強，你不需要寫很複雜的提示詞。用自然的語言描述你想要的效果，就能得到很好的結果。

**範例：**
- ✅ 「把背景換成海邊」
- ✅ 「讓這個人笑得更開心」
- ✅ 「改成水彩畫風格」

不需要：
- ❌ 複雜的技術參數
- ❌ 冗長的描述
- ❌ 專業術語
</Tip>

## Nano Banana 的核心優勢

<AccordionGroup>
  <Accordion title="1. 前所未有的角色一致性" icon="user-check">
    **什麼是角色一致性？**
    
    在多次生成、跨場景、換風格的編輯中，維持同一個人物或物件的外觀細節。
    
    **為什麼重要？**
    - 動畫分鏡製作
    - IP 角色設計
    - 影片製作
    - 系列圖片創作
    - 品牌視覺一致性
    
    這讓 AI 圖像從「單張驚艷」真正走向「連續敘事」。
  </Accordion>
  
  <Accordion title="2. 強大的自然語言理解" icon="comments">
    **特色：**
    - 不需要複雜的提示詞
    - 用日常語言就能精確表達
    - 理解上下文和意圖
    - 支援中文輸入
    
    **實際體驗：**
    你可以說「讓背景更溫暖一點」，它就能理解並調整色調。
  </Accordion>
  
  <Accordion title="3. 精確的局部編輯" icon="pen-to-square">
    **功能：**
    - 指哪改哪
    - 保持其他區域不變
    - 維持光影一致性
    - 自動融合新元素
    
    **應用：**
    - 修改服裝
    - 更換背景
    - 調整表情
    - 添加物件
  </Accordion>
  
  <Accordion title="4. 靈活的風格轉換" icon="palette">
    **能力：**
    - 寫實照片 → 插畫風格
    - 現代風格 → 復古風格
    - 2D 圖像 → 3D 效果
    - 保持主體特徵
    
    **用途：**
    - 快速風格測試
    - 品牌視覺探索
    - 創意發想
    - 客戶提案
  </Accordion>
</AccordionGroup>

## 常見問題

<AccordionGroup>
  <Accordion title="Nano Banana 是免費的嗎？" icon="money-bill">
    **答案：**
    
    - **Google AI Studio**：提供免費額度，適合個人使用和學習
    - **Vertex AI**：按使用量付費，提供企業級功能
    - **第三方平台**：各平台定價不同，部分提供免費試用
    
    **建議：**
    新手可以從 Google AI Studio 的免費額度開始。
  </Accordion>
  
  <Accordion title="需要很強的技術背景嗎？" icon="graduation-cap">
    **答案：不需要！**
    
    Nano Banana 的設計理念就是讓一般人也能輕鬆使用：
    - 用自然語言描述即可
    - 介面直觀易懂
    - 不需要程式背景
    - 不需要複雜的提示詞
    
    **但是：**
    如果要進行 API 整合或企業級應用，需要一定的技術知識。
  </Accordion>
  
  <Accordion title="生成的圖片可以商用嗎？" icon="briefcase">
    **重要提醒：**
    
    1. **確認授權條款**
       - 查看 Google 的使用條款
       - 了解商業使用限制
    
    2. **注意版權問題**
       - 避免包含知名品牌
       - 避免使用明星肖像
       - 注意 IP 相關內容
    
    3. **保險做法**
       - 用於內部測試和概念圖較安全
       - 正式商用前諮詢法律意見
       - 考慮購買企業版授權
  </Accordion>
  
  <Accordion title="與 ChatGPT 的圖片生成有什麼不同？" icon="circle-question">
    **主要差異：**
    
    | 特性 | Nano Banana | ChatGPT (DALL-E) |
    |------|-------------|------------------|
    | **角色一致性** | 極強 | 中等 |
    | **局部編輯** | 精確 | 一般 |
    | **對話理解** | 優秀 | 優秀 |
    | **整合性** | Google 生態 | OpenAI 生態 |
    
    **選擇建議：**
    - 需要角色一致性 → Nano Banana
    - 需要對話式創作 → 兩者都可以
    - 已在使用 Google 服務 → Nano Banana
    - 已在使用 ChatGPT → 可以試試看
  </Accordion>
</AccordionGroup>
