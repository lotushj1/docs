### 進階編輯技巧

<Tabs>
  <Tab title="多層次編輯">
    **概念：**
    分階段進行編輯，每次只調整一個層面。
    
    **編輯順序：**
    
    <Steps>
      <Step title="第一層：主體">
        調整核心主體
        - 人物/產品本身
        - 主要特徵
        - 基礎姿勢
      </Step>
      
      <Step title="第二層：環境">
        修改周邊環境
        - 背景場景
        - 前景元素
        - 空間配置
      </Step>
      
      <Step title="第三層：氛圍">
        調整整體氛圍
        - 光線效果
        - 色調調整
        - 氣氛營造
      </Step>
      
      <Step title="第四層：細節">
        精修細節
        - 質感優化
        - 細部調整
        - 最終潤飾
      </Step>
    </Steps>
    
    **優勢：**
    - 更精確的控制
    - 降低出錯率
    - 容易追蹤修改
    - 可以回溯調整
  </Tab>
  
  <Tab title="參考圖活用">
    **使用參考圖的技巧：**
    
    **1. 風格參考**
    ```
    參考這張圖的風格
    保持主體內容不變
    只學習視覺風格
    ```
    
    **2. 構圖參考**
    ```
    參考這張圖的構圖
    保持相似的視角和配置
    但改變具體內容
    ```
    
    **3. 光線參考**
    ```
    模仿這張圖的光線效果
    相同的光源方向和強度
    營造類似的氛圍
    ```
    
    **4. 色調參考**
    ```
    使用這張圖的色彩方案
    保持相似的色調和對比
    創造協調的配色
    ```
  </Tab>
  
  <Tab title="精確描述技巧">
    **更精確的指令方式：**
    
    **1. 具體位置描述**
    ```
    ❌ 加一棵樹
    ✅ 在畫面左後方加一棵松樹
    
    ❌ 改變顏色
    ✅ 把前景的沙發改成深藍色
    ```
    
    **2. 量化描述**
    ```
    ❌ 讓光線更亮
    ✅ 增加 30% 的亮度
    
    ❌ 加一些人
    ✅ 在背景加 2-3 個路人
    ```
    
    **3. 參考比較**
    ```
    像日落時的暖色調
    類似咖啡廳的溫馨氛圍
    參考北歐極簡風格
    ```
    
    **4. 情緒描述**
    ```
    營造輕鬆愉快的氛圍
    表現專業嚴謹的感覺
    呈現溫馨浪漫的情境
    ```
  </Tab>
</Tabs>

## 常見挑戰與解決方案

<AccordionGroup>
  <Accordion title="角色一致性偏差" icon="user-xmark">
    **問題：**
    經過多次編輯後，角色特徵開始改變
    
    **解決方案：**
    1. **定期回到基準版本**
       - 保存最佳版本作為參考
       - 每隔幾次編輯就重新開始
    
    2. **使用參考圖**
       - 每次編輯都附上原始角色圖
       - 明確指出要保持的特徵
    
    3. **減少連續編輯次數**
       - 一次改動不要太多
       - 分段處理
  </Accordion>
  
  <Accordion title="細節不如預期" icon="magnifying-glass">
    **問題：**
    生成的細節與想像不符
    
    **解決方案：**
    1. **更具體的描述**
       - 詳細說明想要的細節
       - 使用參考圖
    
    2. **後期精修**
       - AI 生成基礎
       - Photoshop 修正細節
    
    3. **多次嘗試**
       - 生成多個版本
       - 選擇最佳結果
       - 組合優點
  </Accordion>
  
  <Accordion title="風格不統一" icon="palette">
    **問題：**
    系列圖片風格有差異
    
    **解決方案：**
    1. **建立風格指南**
       - 記錄成功的參數
       - 固定風格描述詞
       - 統一後製流程
    
    2. **使用範本**
       - 基於成功案例
       - 只改變必要元素
       - 保持其他設定
    
    3. **批次處理**
       - 同時處理相關圖片
       - 保持當下的理解一致
  </Accordion>
  
  <Accordion title="商用版權疑慮" icon="scale-balanced">
    **問題：**
    不確定是否可商業使用
    
    **解決方案：**
    1. **確認授權條款**
       - 閱讀 Google 的使用條款
       - 了解商業使用限制
    
    2. **避免風險內容**
       - 不使用知名品牌
       - 避免明星肖像
       - 不侵犯 IP
    
    3. **保險做法**
       - 內部使用較安全
       - 概念圖和提案用途
       - 正式商用前諮詢法律
       - 考慮購買企業授權
  </Accordion>
</AccordionGroup>
