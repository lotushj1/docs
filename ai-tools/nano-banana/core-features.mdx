## 核心功能總覽

Nano Banana 之所以能在眾多 AI 圖像生成工具中脫穎而出，關鍵在於它解決了過去 AI 繪圖的核心痛點，並提供了前所未有的功能體驗。

## 殺手級應用：角色一致性

### 什麼是角色一致性？

**定義：**在多次生成、跨場景、換風格的編輯中，維持同一個人物或物件的外觀細節。

**為什麼這麼重要？**

<Tabs>
  <Tab title="動畫製作">
    **應用：**
    - 分鏡圖製作
    - 角色設定圖
    - 場景測試
    
    **價值：**
    傳統上需要多位畫師協作並反覆校正，現在 AI 可以自動保持一致性。
  </Tab>
  
  <Tab title="IP 設計">
    **應用：**
    - 品牌角色開發
    - 系列產品設計
    - 周邊商品視覺
    
    **價值：**
    快速探索不同應用場景，確保品牌視覺一致性。
  </Tab>
  
  <Tab title="影片製作">
    **應用：**
    - YouTube 封面系列
    - 限時動態素材
    - 影片縮圖
    
    **價值：**
    建立可辨識的視覺風格，提升品牌識別度。
  </Tab>
  
  <Tab title="故事創作">
    **應用：**
    - 繪本插圖
    - 漫畫分鏡
    - 小說配圖
    
    **價值：**
    從單張圖片走向連續敘事的可能性。
  </Tab>
</Tabs>

### 角色一致性的實際應用

<AccordionGroup>
  <Accordion title="應用一：跟風熱門電影或動畫" icon="film">
    **概念：**
    將品牌角色或產品融入熱門電影、動畫的風格或場景中。

    - 迪士尼風格的吉祥物設計

    - 動畫話題高峰
    - 社群行銷活動
    
    **注意事項：**
    - 避免直接使用受版權保護的角色
    - 以「風格致敬」為主
    - 不要侵犯原作 IP
  </Accordion>
  
  <Accordion title="應用二：節慶主題變化" icon="calendar">
    **台灣熱門節慶：**
    
    1. **農曆新年**

       - 生肖主題

       - 月亮和兔子
       - 烤肉場景
       - 溫馨團圓氛圍
    
    3. **端午節**
       - 粽子元素
       - 龍舟競賽
       - 傳統色調
    
    4. **情人節**
       - 浪漫粉色系
       - 愛心元素
       - 甜蜜氛圍
    
    **操作方式：**
    保持品牌角色不變，只改變背景和節慶元素。
  </Accordion>
  
  <Accordion title="應用三：多場景系列圖" icon="images">
    **系列創作：**
    
    - **旅遊系列**：同一角色在不同景點
    - **四季變化**：同一場景的春夏秋冬
    - **時間流逝**：早晨、中午、黃昏、夜晚
    - **情境展示**：工作、休閒、運動、睡眠
    
    **技巧：**
    1. 先確定核心角色
    2. 用第一張圖作為參考
    3. 每次只改變場景元素
    4. 保持光線風格一致
  </Accordion>

<Note>
**角色一致性的侷限**

雖然 Nano Banana 的角色一致性已經非常強大，但在某些情況下仍可能出現偏差：

- **複雜人物組合**：多人同框時穩定性較低
- **極端角度變化**：從正面轉到背面可能有差異
- **細微特徵**：如痣、皺紋等微小細節可能不穩定
- **連續多次編輯**：編輯次數過多可能累積誤差

**建議：**
- 關鍵角色單獨處理
- 避免一次改變太多
- 定期檢查並回到穩定版本
</Note>

## 指哪修哪：局部編輯功能

### 什麼是局部編輯（Inpainting）？

用最簡單的語言（如：「把沙發換成藍色」）精確修改畫面局部，無需複雜的遮罩操作。

### 局部編輯的核心優勢

<CardGroup cols={2}>
  <Card title="精確控制" icon="bullseye">
    只修改指定區域，其他部分完全不變
  </Card>
  
  <Card title="自然融合" icon="wand-magic-sparkles">
    新元素自動與原畫面融合，保持光影一致
  </Card>
  
  <Card title="簡單指令" icon="comment">
    用自然語言描述，不需要技術操作
  </Card>
  
  <Card title="細節保持" icon="eye">
    材質紋理、光影、皺折都能保持穩定
  </Card>
</CardGroup>

### 局部編輯實戰應用

<Tabs>
  <Tab title="服裝修改">
    **常見需求：**
    - 更換服裝顏色
    - 改變服裝款式
    - 添加配件
    - 修改材質
    
    **範例指令：**
    ```
    把這件上衣改成紅色
    換成正式西裝
    加上一條圍巾
    改成牛仔材質
    ```
    
    **注意事項：**
    - 光影和皺折會自動調整
    - 保持原有的身體姿勢
    - 材質細節會很真實
  </Tab>
  
  <Tab title="背景替換">
    **應用場景：**
    - 產品情境圖製作
    - 人物背景更換
    - 場景氛圍調整
    
    **範例指令：**
    ```
    把背景改成咖啡廳
    換成海邊日落
    改成現代辦公室
    變成下雪的街道
    ```
    
    **效果特點：**
    - 主體邊緣自然融合
    - 光線方向會配合背景
    - 氛圍整體協調
  </Tab>
  
  <Tab title="表情調整">
    **可調整的表情：**
    - 笑容程度
    - 眼神方向
    - 表情變化
    - 情緒轉換
    
    **範例指令：**
    ```
    讓他笑得更開心
    改成認真的表情
    看向鏡頭
    加上驚訝的表情
    ```
    
    **細節保持：**
    - 臉部特徵不變
    - 皮膚質感保持
    - 光影自然調整
  </Tab>
  
  <Tab title="物件添加/移除">
    **添加元素：**
    ```
    在桌上加一杯咖啡
    戴上太陽眼鏡
    手上拿一本書
    旁邊放一盆植物
    ```
    
    **移除元素：**
    ```
    移除背景的人
    拿掉桌上的雜物
    去掉電線桿
    清除水印
    ```
    
    **技巧：**
    - 描述要清楚具體
    - 新物件位置要明確
    - 考慮光線和陰影
  </Tab>
</Tabs>

<Tip>
**局部編輯的最佳實踐**

1. **明確描述位置**
   - ✅ 「把左邊的沙發改成藍色」
   - ❌ 「把沙發改成藍色」（如果有多個沙發）

2. **一次改一處**
   - ✅ 先改顏色，再改材質
   - ❌ 同時改多個屬性（可能混亂）

3. **保持風格一致**
   - 新添加的元素要符合整體風格
   - 考慮光線方向和色調

4. **檢查細節**
   - 放大查看邊緣融合
   - 確認光影自然
   - 驗證材質合理
</Tip>

## 風格魔法：風格轉換功能

### 風格轉換是什麼？

在保持圖片主體內容和結構的前提下，改變整體的視覺風格。

### 支援的風格類型

<Tabs>
  <Tab title="寫實 ↔ 插畫">
    **轉換方向：**
    - 照片 → 手繪插畫
    - 照片 → 數位繪畫
    - 照片 → 水彩風格
    - 照片 → 油畫風格
    - 插畫 → 照片寫實
    
    **應用：**
    - 製作繪本風格
    - 藝術化商品照
    - 創意海報設計
    - 社群媒體素材
  </Tab>
  
  <Tab title="現代 ↔ 復古">
    **時代風格：**
    - 現代照片 → 80 年代風格
    - 現代照片 → 黑白老照片
    - 現代照片 → 復古海報
    - 復古照片 → 現代修復
    
    **應用：**
    - 懷舊主題設計
    - 品牌復古活動
    - 時代對比創作
    - 照片修復還原
  </Tab>
  
  <Tab title="2D ↔ 3D">
    **維度轉換：**
    - 平面插圖 → 3D 渲染
    - 照片 → 3D 卡通
    - 素描 → 立體模型感
    
    **應用：**
    - 角色設計預覽
    - 產品 3D 化
    - 遊戲美術參考
    - 動畫概念圖
  </Tab>
  
  <Tab title="特殊藝術風格">
    **藝術流派：**
    - 梵谷星空風格
    - 浮世繪風格
    - 波普藝術
    - 極簡主義
    - 賽博龐克
    - 蒸氣龐克
    
    **應用：**
    - 藝術創作
    - 特色海報
    - 品牌視覺探索
    - 展覽素材
  </Tab>
</Tabs>

### 風格轉換的應用技巧

<Steps>
  <Step title="選擇合適的原圖">
    **理想原圖特性：**
    - 主體清晰
    - 構圖簡潔
    - 光線良好
    - 細節豐富
    
    **避免：**
    - 過於複雜的背景
    - 模糊不清的照片
    - 過度曝光或欠曝
  </Step>
  
  <Step title="明確風格描述">
    **描述要點：**
    - 指定具體風格名稱
    - 說明想要的色調
    - 提及參考藝術家（如適用）
    - 描述期望的氛圍
    
    **範例：**
    ```
    轉換成水彩插畫風格，
    色彩柔和，帶有手繪質感，
    保持溫暖的色調
    ```
  </Step>
  
  <Step title="保持核心元素">
    **在指令中說明：**
    - 哪些元素要保持
    - 哪些可以改變
    - 重要細節的保留
    
    **範例：**
    ```
    改成復古海報風格，
    但保持人物的臉部特徵清晰
    ```
  </Step>
  
  <Step title="迭代優化">
    **優化流程：**
    1. 第一次轉換查看整體效果
    2. 根據結果調整描述
    3. 如需要可再次微調
    4. 比較不同版本
  </Step>
</Steps>

<Warning>
**風格轉換的注意事項**

1. **細節可能損失**
   - 轉換成簡化風格時，細節會減少
   - 某些風格不適合需要精確細節的圖片

2. **風格理解偏差**
   - 同一個風格名稱可能有不同詮釋
   - 建議提供更多描述或參考

3. **版權考量**
   - 避免直接模仿特定藝術家的獨特風格
   - 使用「風格類型」而非「某某畫家風格」

4. **適用性評估**
   - 不是所有圖片都適合所有風格
   - 商業用途要考慮品牌調性
</Warning>

## 智慧理解：邏輯推理能力

### 什麼是邏輯推理？

Nano Banana 不只是執行指令，更能理解背後的邏輯和意圖，做出合理的視覺決策。

### 邏輯推理的實際表現

<AccordionGroup>
  <Accordion title="情境理解" icon="brain">
    **能力：**
    當你說「讓畫面更溫暖」，它會：
    - 調整色溫偏暖
    - 增加暖色調光線
    - 調整陰影色彩
    - 營造溫馨氛圍
    
    而不只是簡單地增加橙色濾鏡。
  </Accordion>
  
  <Accordion title="物理常識" icon="atom">
    **理解：**
    - 光影的合理性
    - 物體的重力
    - 透視的正確性
    - 材質的物理特性
    
    **範例：**
    添加陰影時，會根據光源位置自動計算陰影方向和強度。
  </Accordion>
  
  <Accordion title="文化脈絡" icon="globe">
    **認知：**
    - 節慶的典型元素
    - 場景的慣用配置
    - 服裝的文化意涵
    - 色彩的象徵意義
    
    **範例：**
    說「改成聖誕節氛圍」，會自動添加聖誕樹、裝飾、雪景等相關元素。
  </Accordion>
  
  <Accordion title="美學判斷" icon="palette">
    **能力：**
    - 色彩和諧搭配
    - 構圖平衡
    - 視覺層次
    - 風格統一
    
    **範例：**
    更換背景時，會選擇與主體色調協調的背景色彩。
  </Accordion>
</AccordionGroup>

## 功能組合應用

最強大的使用方式是組合多個功能：

<Steps>
  <Step title="建立角色">
    使用角色一致性功能，創建或導入你的核心角色
  </Step>
  
  <Step title="調整細節">
    用局部編輯功能微調角色的服裝、表情、姿勢
  </Step>
  
  <Step title="變換場景">
    保持角色不變，更換不同的背景場景
  </Step>
  
  <Step title="風格實驗">
    將同一場景轉換成不同視覺風格
  </Step>
  
  <Step title="系列產出">
    快速生成一系列保持一致性的設計稿
  </Step>
</Steps>

<Tip>
**專業工作流程建議**

1. **建立基礎模板**
   - 先確定核心角色或主視覺
   - 測試並固定最佳版本
   - 作為後續所有變化的基準

2. **分層處理**
   - 先處理主體
   - 再調整背景
   - 最後做風格化

3. **版本管理**
   - 保存每個重要的中間版本
   - 標記成功的參數設定
   - 建立可重複使用的模板

4. **批次處理**
   - 相同類型的任務一起處理
   - 利用成功的模式
   - 提升整體效率
</Tip>

## 功能限制與解決方案

<AccordionGroup>
  <Accordion title="複雜場景的穩定性" icon="triangle-exclamation">
    **問題：**
    多人物、複雜背景時，一致性可能降低
    
    **解決方案：**
    - 分開處理主體和背景
    - 減少單次修改的複雜度
    - 使用更明確的描述
  </Accordion>
  
  <Accordion title="極端角度變化" icon="rotate">
    **問題：**
    從正面轉到背面或側面時，細節可能改變
    
    **解決方案：**
    - 避免過大的角度跳躍
    - 用中間角度作為過渡
    - 關鍵特徵多次確認
  </Accordion>
  
  <Accordion title="文字生成" icon="font">
    **問題：**
    AI 生成的文字仍可能出錯
    
    **解決方案：**
    - 重要文字用後製加上
    - 或用 Photoshop 等工具修正
    - 簡短文字成功率較高
  </Accordion>
  
  <Accordion title="手部細節" icon="hand">
    **問題：**
    手指數量或姿勢可能不準確
    
    **解決方案：**
    - 避免特寫手部
    - 用道具遮擋部分手部
    - 後期手動修正
    - 或重新生成直到滿意
  </Accordion>
</AccordionGroup>
