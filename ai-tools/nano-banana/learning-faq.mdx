## Nano Banana 的核心優勢

在開始學習之前，先了解 Nano Banana 的核心優勢能幫助你更有方向地學習：

<AccordionGroup>
  <Accordion title="1. 前所未有的角色一致性" icon="user-check">
    **什麼是角色一致性？**
    
    在多次生成、跨場景、換風格的編輯中，維持同一個人物或物件的外觀細節。
    
    **為什麼重要？**
    - 動畫分鏡製作
    - IP 角色設計
    - 影片製作
    - 系列圖片創作
    - 品牌視覺一致性
    
    這讓 AI 圖像從「單張驚艷」真正走向「連續敘事」。
  </Accordion>
  
  <Accordion title="2. 強大的自然語言理解" icon="comments">
    **特色：**
    - 不需要複雜的提示詞
    - 用日常語言就能精確表達
    - 理解上下文和意圖
    - 支援中文輸入
    
    **實際體驗：**
    你可以說「讓背景更溫暖一點」，它就能理解並調整色調。
  </Accordion>
  
  <Accordion title="3. 精確的局部編輯" icon="pen-to-square">
    **功能：**
    - 指哪改哪
    - 保持其他區域不變
    - 維持光影一致性
    - 自動融合新元素
    
    **應用：**
    - 修改服裝
    - 更換背景
    - 調整表情
    - 添加物件
  </Accordion>
  
  <Accordion title="4. 靈活的風格轉換" icon="palette">
    **能力：**
    - 寫實照片 → 插畫風格
    - 現代風格 → 復古風格
    - 2D 圖像 → 3D 效果
    - 保持主體特徵
    
    **用途：**
    - 快速風格測試
    - 品牌視覺探索
    - 創意發想
    - 客戶提案
  </Accordion>
</AccordionGroup>

## 常見問題 FAQ

<AccordionGroup>
  <Accordion title="Nano Banana 是免費的嗎？" icon="money-bill">
    **答案：**
    
    - **Google AI Studio**：提供免費額度，適合個人使用和學習
    - **Vertex AI**：按使用量付費，提供企業級功能
    - **第三方平台**：各平台定價不同，部分提供免費試用
    
    **建議：**
    新手可以從 Google AI Studio 的免費額度開始。
  </Accordion>
  
  <Accordion title="需要很強的技術背景嗎？" icon="graduation-cap">
    **答案：不需要！**
    
    Nano Banana 的設計理念就是讓一般人也能輕鬆使用：
    - 用自然語言描述即可
    - 介面直觀易懂
    - 不需要程式背景
    - 不需要複雜的提示詞
    
    **但是：**
    如果要進行 API 整合或企業級應用，需要一定的技術知識。
  </Accordion>
  
  <Accordion title="生成的圖片可以商用嗎？" icon="briefcase">
    **重要提醒：**
    
    1. **確認授權條款**
       - 查看 Google 的使用條款
       - 了解商業使用限制
    
    2. **注意版權問題**
       - 避免包含知名品牌
       - 避免使用明星肖像
       - 注意 IP 相關內容
    
    3. **保險做法**
       - 用於內部測試和概念圖較安全
       - 正式商用前諮詢法律意見
       - 考慮購買企業版授權
  </Accordion>
  
  <Accordion title="與 ChatGPT 的圖片生成有什麼不同？" icon="circle-question">
    **主要差異：**
    
    | 特性 | Nano Banana | ChatGPT (DALL-E) |
    |------|-------------|------------------|
    | **角色一致性** | 極強 | 中等 |
    | **局部編輯** | 精確 | 一般 |
    | **對話理解** | 優秀 | 優秀 |
    | **整合性** | Google 生態 | OpenAI 生態 |
    
    **選擇建議：**
    - 需要角色一致性 → Nano Banana
    - 需要對話式創作 → 兩者都可以
    - 已在使用 Google 服務 → Nano Banana
    - 已在使用 ChatGPT → 可以試試看
  </Accordion>
  
  <Accordion title="學習 Nano Banana 需要多久？" icon="clock">
    **時間規劃：**
    
    - **基礎操作**：1-2 天
    - **熟練使用**：1-2 週
    - **進階應用**：1 個月
    - **專業整合**：2-3 個月
    
    **加速學習的方法：**
    - 每天實際操作 30 分鐘
    - 參考優秀案例
    - 加入社群交流
    - 記錄學習心得
  </Accordion>
  
  <Accordion title="如何提升生成品質？" icon="star">
    **5 個關鍵技巧：**
    
    1. **提示詞要具體**
       - 詳細描述場景、光線、風格
       - 使用專業術語
    
    2. **善用參考圖**
       - 上傳參考圖片
       - 說明想要的部分
    
    3. **迭代優化**
       - 不要期待一次完美
       - 持續對話調整
    
    4. **學習成功案例**
       - 研究優秀作品的提示詞
       - 建立自己的模板庫
    
    5. **結合後製**
       - 用 Photoshop 精修細節
       - AI 生成 + 人工優化 = 最佳效果
  </Accordion>
  
  <Accordion title="遇到問題時該怎麼辦？" icon="circle-info">
    **解決問題的步驟：**
    
    1. **查閱文檔**
       - 先看本指南的相關章節
       - 查看官方文檔
    
    2. **搜尋社群**
       - Google 搜尋相關問題
       - 查看 Reddit、Discord 討論
    
    3. **實驗測試**
       - 嘗試不同的描述方式
       - 簡化要求再逐步增加
    
    4. **尋求幫助**
       - 加入 AI 設計社群
       - 在論壇發問
       - 參考他人經驗
  </Accordion>
</AccordionGroup>
